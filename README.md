Pison Data Cleaning Helper Tools

Cloning the project
- Create GitHub account
- Install VSCode
- Copy repository link from "Code" button
- Paste into VSCode

Running the project:
- Install any version of Python
- In the terminal, run "pip install numpy" "pip install pandas"
- Create virtual environment

General use:
- Move data you want to clean into data file
- Run main
- Enter file location (ex. data/all_data.csv)
- Enter test type you want (r/readiness, a/agility, f/focus)
- Create a new file with specified name
